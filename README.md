SafeSpend

Description 
A budget tracking web application that help users manage their income, expense, and financial goals with real-time insight and analytics.

 🚀 Features
- User authentication
- Expense tracking
- Budget goals
- Analytics dashboard and calander
- Smart Insights

## 🔒 Security Features
- CSRF protection
- Input validation
- Rate limiting

## 🛠️ Tech Stack
- PHP
- MySQL
- HTML/CSS

## ▶️ How to Run
1. Install XAMPP
2. Import database.sql
3. Run localhost/project
